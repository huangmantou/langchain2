## 穿搭推荐指南

### 春季穿搭（3-5月）
#### 通勤穿搭
- 男士：牛津纺衬衫 + 九分裤 + 帆布鞋，外搭轻薄针织开衫
- 女士：亚麻衬衫 + 阔腿裤 + 乐福鞋，或连衣裙 + 薄款风衣
- 色彩建议：浅蓝、米白、卡其、淡粉等柔和色调

#### 休闲穿搭
- 男士：UT印花T恤 + 牛仔裤 + 运动鞋
- 女士：宽松T恤 + 高腰半裙 + 小白鞋
- 叠穿技巧：早晚温差大时，T恤外搭薄款卫衣或衬衫

### 夏季穿搭（6-8月）
#### 清凉通勤
- 男士：AIRism Polo衫 + 弹力九分裤 + 凉鞋
- 女士：AIRism blouse + 棉麻阔腿裤 + 平底鞋
- 面料选择：优先AIRism、亚麻、纯棉等透气面料

#### 度假/户外
- 防晒搭配：UPF50+防晒衣 + 短裤/短裙 + 遮阳帽
- 海边穿搭：宽松亚麻衬衫 + 泳装 + 拖鞋
- 运动穿搭：DRY-EX速干T恤 + 运动短裤 + 跑鞋

### 秋季穿搭（9-11月）
#### 层次叠穿
- 男士：T恤 + 法兰绒衬衫（敞开） + 牛仔裤
- 女士：打底衫 + 针织马甲 + 半裙 + 短靴
- 叠穿公式：内薄外厚，内浅外深

#### 商务休闲
- 男士：HEATTECH内衣 + 衬衫 + 休闲西装 + 锥形裤
- 女士：高领打底 + 西装外套 + 直筒裤 + 尖头鞋

### 冬季穿搭（12-2月）
#### 保暖通勤
- 男士：HEATTECH极暖款 + 羊毛衫 + 羽绒服 + 弹力裤
- 女士：HEATTECH高暖款 + 针织连衣裙 + 长大衣 + 靴子
- 保暖层次：内层吸湿（HEATTECH）→ 中层保暖（羊毛/抓绒）→ 外层防风（羽绒服/大衣）

#### 极寒穿搭
- 上身：HEATTECH高暖款 + 羽绒背心 + 长款羽绒服
- 下身：HEATTECH裤袜 + 加绒裤 + 防风水裤
- 配件：羊绒围巾 + 触屏手套 + 毛线帽

### 场景穿搭建议
#### 面试/正式场合
- 男士：白色/浅蓝衬衫 + 深色西裤 + 皮鞋，可加领带
- 女士：白色衬衫 + 西装裤/A字裙 + 中跟鞋，妆容简洁
- 原则：简洁大方，避免过于花哨的颜色和图案

#### 约会
- 男士：纯色T恤/衬衫 + 修身牛仔裤 + 小白鞋，干净整洁最重要
- 女士：修身针织衫 + 高腰半裙 + 短靴，或连衣裙 + 外套
- 色彩建议：暖色调（米色、驼色、酒红）更显亲和

#### 旅行
- 核心原则：舒适、百搭、易打理、可叠穿
- 建议携带：2-3件上衣可互相搭配，1条百搭裤装，1件外套应对温差
- 面料选择：抗皱、快干、弹性好的面料

### 色彩搭配基础
- 同色系搭配：深浅不同的同色搭配，高级感强（如浅蓝+深蓝）
- 邻近色搭配：色轮上相邻的颜色搭配，和谐自然（如蓝+绿）
- 对比色搭配：色轮上相对的颜色搭配，视觉冲击力强（如蓝+橙）
- 万能配色：黑、白、灰、牛仔蓝可与任何颜色搭配
- 全身不超过3个颜色（不含黑白灰），避免过于杂乱
